READ ME

To install these tools in your local environment, the following steps are required:

1. Create a MySQL database on your server named 'tools'
2. Run the code included in the file 'table_create.sql' located in the installation
   directory to create and populate the necessary tables.
3. Set the following variables for your own system in the 'config.php' file
   located in the installation directory:
   - BASE_URL
   - LINK_ROOT
   - DB_HOST
   - DB_NAME
   - DB_PORT
   - DB_USER
   - DB_PASS
   Details on the setting of these variables is included in the comments in
   installation/config.php
